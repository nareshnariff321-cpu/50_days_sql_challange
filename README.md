This project is a part of my 50 Days SQL Challenge, where I solved real-world business problems using SQL queries and relational database concepts.
The database contains 5 interconnected tables: Customers, Orders, Order Details, Products, and Payments, designed using an ERD structure to simulate a real e-commerce/business environment.
In this project,
I worked on solving analytical business questions such as:
 1)Top revenue-generating products
 2)Highest sales month and top revenue day
 3)Average orders and customer spending analysis
 4)Revenue generated per customer and per product
 5) Order trends and business performance insights
Through this challenge, I practiced:
Joins and Relationships
Aggregate Functions
GROUP BY and HAVING
Subqueries and CTEs
Real-time business data analysis using SQL
This project helped me improve my problem-solving skills and gain hands-on experience in writing efficient SQL queries for real-world scenarios.
